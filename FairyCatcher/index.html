Fairy Catcher

Instructions

Use WASD or the arrow keys to move the character, and the spacebar to switch between dimensions. Fairies will randomly appear in the dimensions. Walking over the fairies will capture them. Try and catch as many as you can!

This is an arcade game where you have 60 seconds to capture as many fairies as possible. Switch between dimensions to capture all the fairies. Developed in Java using the LibGDX framework.
