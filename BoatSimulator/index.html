Boat Simulator

Instructions

Use WASD or the arrow keys to move through the water.

Pressing S or the down arrow key will submerge the boat in the water.

Pressing W or the up arrow key will let the boat jump.

The boat can do a double jump if you press W/up again.

This is a sandbox where you can move around in a boat in buoyant water. Developed in Java using the LibGDX and Box2D framework.
